# Android Local Flavor Scaffold

This directory is intentionally a scaffold only.

Do not add a real `google-services.json` here until the local or emulator-facing
native flavor is activated on Android.

Future contents can include:

- `google-services.json`
- `AndroidManifest.xml`
- flavor-specific icons or resources

Planned Android publication IDs:

- local: `org.adfoot.app.local`
- staging: `org.adfoot.app.staging`
- production: `org.adfoot.app`
